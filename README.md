В ходе работы было проведено модульное тестирование математических функций, вынесенных в отдельный класс MathFunctions. Для проверки корректности работы были разработаны три теста, охватывающие основные вычисления:

- вычисление сложного математического выражения;

- проверка работы функции sinh(x);

- проверка логики вычисления результата с условиями.


Все тесты были успешно выполнены.

Причина успешного выполнения тестов:

- математическая логика была вынесена из пользовательского интерфейса в отдельный класс, что позволило корректно её тестировать;

- входные данные в тестах заданы корректно и соответствуют допустимым значениям;

- методы реализованы без ошибок и возвращают ожидаемые результаты;

- в тестах используются проверки (Assert), которые точно сравнивают ожидаемые и фактические значения.
