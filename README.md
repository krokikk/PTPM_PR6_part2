# Практическая работа №4. Поддержка и тестирование программных модулей
## Тестирование «белым ящиком»
## Коняева София 3ИСИП-523. Вариант 5

### Приложение для расчёта математических функций с реализацией навигации, проверки входных данных и тестирования методом «белого ящика».
Приложение реализует ввод параметров, вычисление функций, вывод результатов и построение графика.

---
### Цель работы
Получение практических навыков тестирования методом белого ящика при разработке WPF-приложения.

### Страница 3

Расчёт функции:

<img width="462" height="75" alt="image" src="https://github.com/user-attachments/assets/671a0c68-ba81-405a-8811-caf82d70fa66" />

Пользователь вводит:
- x₀ – начальное значение
- xk – конечное значение

Приложение:
- Выполняет расчёт в цикле
- Выводит список всех значений x и y

Дополнительно:
- Проверка корректности шага dx
